Clothing E-Commerce App

A full-featured clothing e-commerce mobile application built with Flutter, designed with scalable architecture and real-world use cases in mind.

---

🚀 Overview

This project demonstrates how to build a production-ready e-commerce application using clean architecture principles, proper state management, and a modular code structure.

The focus is not only on UI, but on maintainability, scalability, and realistic application behavior.

---

🧱 Architecture

The app is built using:

- MVVM (Model - View - ViewModel)
- Cubit (Bloc) for State Management

This ensures:

- Clear separation of concerns
- Maintainable and testable code
- Scalable structure for future expansion

---

✨ Features

- Product listing with categories and filtering
- Favorites (Wishlist) system
- Shopping Bag (Cart) management
- Orders history screen
- Map integration using OpenStreetMap
- Image upload with stored image URL
- Dark & Light mode support
- Persistent theming using SharedPreferences

---

🔐 Authentication & Backend (Firebase)

- User Authentication using Firebase (Login / Register)
- Email Verification system
- Forgot Password functionality
- All user data handled securely through Firebase
- Cloud Firestore used for storing application data (products, orders, user-related data)

---

⚙️ Technical Highlights

- Clean and modular code structure
- Reusable widgets and components
- Efficient state management using Cubit
- Local data persistence for improved user experience
- Scalable backend integration using Firebase services
